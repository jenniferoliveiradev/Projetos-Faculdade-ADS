# Matrículas em Cursos Online

Sistema simples para matrícula em cursos online.

## Estrutura

- `backend/` - Servidor Express (Node.js)
- `frontend/` - Interface Vue.js

## Instalação

### Backend
```
cd backend
npm install
npm start
```
Rodará em http://localhost:5000

### Frontend
```
cd frontend
npm install
npm run dev
```
Rodará em http://localhost:5173

## Funcionamento

1. Acesse o frontend no navegador
2. Escolha um curso da lista
3. Preencha nome e email
4. Confirme a matrícula
5. Converse confirmação com ID da matrícula
