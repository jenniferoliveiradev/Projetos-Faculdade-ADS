SISTEMA DE ATENDIMENTO - CLINICA MEDICA

Inicio rapido (Windows):
1. Abra um terminal na raiz do projeto.
2. Execute `start.bat`.
3. Aguarde abrir as duas janelas (backend e frontend).
4. Acesse `http://localhost:3000`.

Inicio manual:

Backend:
  cd backend
  npm install
  copy .env.example .env
  npm start

Frontend (outro terminal):
  cd frontend
  npm install
  npm run dev

Verificacao:
- Backend: `http://localhost:5000/api/teste`
- Frontend: `http://localhost:3000`

Admin:
- O cadastro pela tela cria usuario paciente.
- Para promover admin:
  cd backend
  npm run create-admin -- --email admin@clinica.com
